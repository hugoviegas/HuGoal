«include»Concluir treino inclui Ganhar XP por ação
«include»Identificar por foto inclui Sistema IA estimar macros
«include»Gerar dieta IA inclui Definir metas nutricionais

«extend»Desbloquear conquista pode estender Criar post (conquista)
«extend»Adicionar alimento texto estende Identificar por foto (fallback)
«extend»Gerar treino com IA estende Criar ficha de treino
«generalization»Usuário Pro herda todos os casos de uso do Usuário Free
«generalization»Personal Trainer herda Usuário Free + casos exclusivos de PT
